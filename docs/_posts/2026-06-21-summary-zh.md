## [SMPTE 免费开放其标准](https://www.smpte.org/blog/smpte-makes-its-standards-freely-accessible-openingstandards-library-to-the-global-media-technology-community) ⭐️ 5.0/10

SMPTE 宣布将其媒体技术标准免费向全球开放，并通过基于 GitHub 的工作流程和 HTML 编写方式实现开发现代化。 此举降低了媒体技术专业人士和初创企业的门槛，可能加速媒体制作和分发领域的创新。同时，它使 SMPTE 与开放标准运动保持一致，可能提高其标准的采用率。 该举措包括采用 GitHub 进行版本控制和问题跟踪，过渡到结构化 HTML 编写，并实施集成发布管道。超过 800 项 SMPTE 标准现已免费提供。

hackernews · zdw · Jun 20, 17:01 · [社区讨论](https://news.ycombinator.com/item?id=48610827)

**背景**: SMPTE（电影与电视工程师协会）是一个国际公认的标准组织，已为媒体技术行业制定了 800 多项标准。此前，获取这些标准需要付费，这限制了它们的使用，尤其是对于小型实体。此次开放获取的举措效仿了 IETF 等成功模式，后者通过免费标准促进了广泛采用和创新。

## [Cloudflare 推出 AI 代理临时账户](https://blog.cloudflare.com/temporary-accounts/) ⭐️ 5.0/10

Cloudflare 宣布为 AI 代理推出临时账户，允许临时部署 Worker，持续 60 分钟，并可认领为永久账户。该功能可通过 Wrangler CLI 的--temporary 标志使用。 该功能降低了开发者和 AI 代理快速测试和部署代码的门槛，无需永久设置，可能加速 Cloudflare Workers 的采用。然而，缺乏硬性计费上限仍是对成本敏感用户的担忧。 临时部署在 60 分钟后过期，除非被认领，Cloudflare 还应用了速率限制和滥用预防检查以防止滥用。该功能专为 AI 代理、PR 预览和代码审查场景设计。

hackernews · farhadhf · Jun 20, 11:19 · [社区讨论](https://news.ycombinator.com/item?id=48608394)

**背景**: 临时环境是用于测试、预发布和短期任务的临时云原生部署。Cloudflare Workers 是一个在边缘运行代码的无服务器平台。社区长期以来一直要求硬性计费上限以避免意外成本，但 Cloudflare 尚未实施。
