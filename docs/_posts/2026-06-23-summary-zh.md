## [2025 年 Linux 安全启动证书到期需用户手动更新](https://lwn.net/Articles/1029767/) ⭐️ 5.0/10

由微软签名的 Linux 安全启动证书将于 2025 年到期，用户需更新系统的安全启动密钥以维持启动完整性。若不操作，系统可能无法启动或显示警告。 这影响所有启用了安全启动的 Linux 用户，可能导致启动失败和安全警告。系统管理员必须规划更新以避免中断。 到期涉及用于签名 shim 引导加载程序的微软证书，该证书随后验证后续启动组件。用户可通过固件更新或使用 mokutil 等工具手动注册新密钥来更新。

hackernews · weaksauce · Jun 22, 18:24 · [社区讨论](https://news.ycombinator.com/item?id=48633941)

**背景**: 安全启动是 UEFI 的一项功能，确保仅签名软件启动，防止恶意软件。Linux 发行版使用一个名为 shim 的小型签名引导加载程序（由微软签名）来链式加载主引导加载程序。shim 证书有有效期，需定期更新。

## [Claude Code 的“扩展思考”是有损摘要，并非真实推理](https://patrickmccanna.net/the-text-in-claude-codes-extended-thinking-output-is-not-authentic/) ⭐️ 5.0/10

一篇博客文章指出，Claude Code 的“扩展思考”输出是模型推理过程的有损摘要，而非真实的思维链，引发了透明度担忧。 这对 Anthropic 等 AI 公司的投资者很重要，因为隐藏推理过程可能掩盖安全风险（如推理期间的提示注入），并可能引发监管对 AI 透明度的关注。 该文章将有损摘要比作将 JPEG 转换为 BMP 再转换回来，导致数据丢失。隐藏推理还使提示优化复杂化，并可能允许攻击者在推理期间通过函数调用窃取数据。

## [博主在摄影师版权案中胜诉](https://blog.ericgoldman.org/archives/2026/06/blogger-defeats-photographers-copyright-claim-sokolskyfilm-v-messiah.htm) ⭐️ 5.0/10

一家联邦法院裁定，博主在博客文章中使用摄影师图片并附评论属于合理使用，即使图片未经修改。 该裁决明确了在线评论中合理使用的边界，可能影响内容创作者和博主未经许可使用受版权保护图片的方式。 法院认为该使用具有转换性，因为博客文章添加了评论，图片并非焦点，且未造成市场损害。该博客文章仅有 43 次浏览量。
